### Core 9-Command Toolkit Documentation
Type `termuxplus` inside your shell to review your utilities panel:
1. **`subfind`** - Lightning-fast subdomain discovery engine.
2. **`editf`** - Quick-launch text configuration editor.
3. **`lsa`** - Enhanced system file visibility (hidden maps).
4. **`nload`** - Streamlined file saving and storage loading.
5. **`copkg`** - Smart package installer featuring size previews (B, KB, MB, GB).
6. **`scan-storage`** - Visualizes real-time storage hardware usage tracking.
7. **`search`** - Instant command string history search.
8. **`plus-toggle`** - Master switch to flip Termux+ features on or off.
9. **`tp-disable`** - The "Danger Zone" uninstaller with verification warnings.
