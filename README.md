### Universal Installation Command
Run this single command inside your Termux terminal to install the package instantly:
```bash
pkg update && pkg install curl -y && curl -L -o termuxplus_1.0.2_all.deb raw.githubusercontent.com/DrOrenTox/TermuxPlus/main/termuxplus_1.0.2_all.deb && pkg install ./termuxplus_1.0.2_all.deb -y
```

### Available Utilities
Once installed, you can call these commands from any folder location:
* `login` - Verifies your system time security layout.
* `csds` - Evaluates physical external SD card memory storage capacities.
* `settings` - Launches your system configuration management utility panel.
