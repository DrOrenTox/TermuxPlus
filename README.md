### Complete 16-Command List Documentation
Once installed, type `termuxplus` or any of these native system words to run them:
1. **`login`** - Local system time security authorization prompt.
2. **`csds`** - External hardware micro SD memory space analyzer.
3. **`settings`** - App update manager menu dashboard.
4. **`info`** - Device metrics, Android core layers, and architecture summary.
5. **`backup`** - Compresses configuration trees to internal backup directories.
6. **`clean`** - Wipes broken cache tracks to free up internal system storage.
7. **`network`** - Verifies server communication speeds and DNS performance.
8. **`theme`** - Installs color configurations for custom layouts.
9. **`weather`** - Fetches current real-time weather forecasts directly via terminal.
10. **`syslock`** - Forces manual password lockout properties on command line entry.
11. **`sysstatus`** - Visualizes real-time metrics for phone processor and RAM loads.
12. **`battery`** - Displays status reports on phone charge capacity and health.
13. **`filemanager`** - Fast shortcut tool to scan, move, or modify file properties.
14. **`uninstaller`** - Clean wipe code to restore default terminal package states.
15. **`webcheck`** - Tests website connectivity responses and header parameters.
16. **`helpme`** - Launches the integrated user manual directory.
